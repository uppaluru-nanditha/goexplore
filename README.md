# GoExplore 🌍

A full stack tourist guide web application built with React.js and FastAPI.

## Live Demo
Coming soon!

## Features
- 🏙️ Browse cities from around the world
- 🔍 Search cities and countries in real time
- ✨ AI-powered city recommendations by category
- ❤️ Save and manage your favorite cities
- 📱 Fully responsive design

## Tech Stack
**Frontend:**
- React.js (Hooks, Context API)
- React Router v6
- CSS3 (Flexbox & Grid)
- LocalStorage for persistent favorites

**Backend:**
- Python & FastAPI
- REST API design
- CORS middleware

## Getting Started

### Frontend
```bash
npm install
npm start
```

### Backend
```bash
cd goexplore-backend
source venv/bin/activate
uvicorn main:app --reload
```
